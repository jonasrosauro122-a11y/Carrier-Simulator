# LAVA Carrier Portal Pro - Standalone Training Simulator

This is a static, standalone Personal Lines carrier portal simulator for VA training. It works directly on GitHub, Netlify, GitHub Pages, or any static hosting service.

## Netlify Setup

- Build command: leave blank
- Publish directory: `.`

## Folder Structure

```text
index.html
css/portal.css
js/app.js
js/rating-engine.js
data/carriers.js
data/questions.js
images/logo.png
images/favicon.png
docs/trainer-guide.md
netlify.toml
_redirects
README.md
```

## Login

- VA Trainee: enter any name and launch.
- Trainer/TL: choose Trainer/TL role and use access code `LAVA2026`.

## Features

- VA login and Trainer/TL login
- Auto and Home new business quote workflows
- Multi-step carrier-style intake
- Multiple vehicle and driver entry for auto
- Underwriting question logic
- Live risk score and QA score
- Simulated carrier rating engine
- Preferred / Standard / Referral / Declined outcomes
- Work queue with draft continuation
- Trainer review checklist and score
- Audit logs and login logs
- CSV and JSON export
- Dark mode
- Print / Save as PDF
- Local browser storage only, no backend required

## Important Training Disclaimer

This is a training simulator only. Premiums, eligibility, underwriting outcomes, carrier names, and appetite behavior are fictional or simplified for learning purposes. Do not enter real customer private information.
