# CarrierOps SOP Guide

## Policy Search

1. Search by policy number first.
2. If no result, search by named insured.
3. Verify address, email, phone, policy term, and status.
4. Select the correct service action.

## Auto ID Card

1. Pull up active Auto policy.
2. Verify named insured, vehicle, VIN, and effective dates.
3. Generate ID card.
4. Download or print/save as PDF.

## Payment

1. Pull up policy.
2. Verify current balance.
3. Enter payment amount, method, payment date, and payer.
4. Add payment authorization remark.
5. Submit and download receipt.

## Endorsement

1. Pull up policy.
2. Choose endorsement type.
3. Confirm effective date.
4. Enter current information and new information.
5. Upload supporting documents if required.
6. Add processor remark.
7. Submit and download endorsement packet.

## Cancellation

1. Pull up policy.
2. Verify requester authority.
3. Confirm cancellation type and effective date.
4. Review proof and refund requirements.
5. Enter cancellation reason.
6. Submit and download cancellation packet.

## Remarketing

1. Pull up policy.
2. Review current premium and risk factors.
3. Select remarketing reason.
4. Add marketing notes.
5. Generate market comparison.
6. Download summary.
