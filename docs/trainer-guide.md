# Trainer Guide

Use this simulator to coach VAs on realistic Personal Lines quoting habits:

1. Confirm applicant details before rating.
2. Verify prior insurance and lapse information.
3. Enter all vehicles, drivers, household members, and property details.
4. Review underwriting questions carefully.
5. Check risk score, QA score, missing documents, and referral flags.
6. Submit files for Trainer/TL review.
7. Use the Trainer Review dashboard to score accuracy and add coaching notes.

The simulator stores all records in the local browser only. Export JSON backups if you need to move records to another device.
